SPRINT 4 — GAMIFICATION & EXPERIENCE
Duration: 1 week
================================================================================

Goal:
  Enhance game feel, engagement, and reward systems.

Key Deliverables:
  ✓ Framer Motion animations
  ✓ Sound system
  ✓ Score system
  ✓ High score (localStorage)
  ✓ Win screen with confetti
  ✓ Star rating (1–5 stars)

Tasks:
  - Add 3D flip, match glow & wrong shake animations via Framer Motion
  - Implement sound system: flip, match, wrong, win + mute toggle
  - Score calculation based on time + moves with wrong-attempt penalties
  - Persist high scores to localStorage, show confetti on completion
  - Star rating system (1–5 stars) for both Fun and Learning modes

Star Rating Logic:

  Fun Mode:
    ★★★★★  Moves ≤ 8,  Time ≤ 1:00
    ★★★★   Moves ≤ 12, Time ≤ 2:00
    ★★★    Moves ≤ 16, Time ≤ 3:00
    ★★     Moves ≤ 20, Time ≤ 4:00
    ★      Any other score

  Learning Mode:
    ★★★★★  Accuracy ≥ 95%, Moves ≤ 8
    ★★★★   Accuracy ≥ 85%, Moves ≤ 12
    ★★★    Accuracy ≥ 75%, Moves ≤ 16
    ★★     Accuracy ≥ 60%, Moves ≤ 20
    ★      Any other score

Acceptance Criteria:
  ✓ Animations run at 60 FPS
  ✓ Sounds synced correctly
  ✓ Scores calculated correctly
  ✓ High scores persist across sessions
